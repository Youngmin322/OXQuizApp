## 주요 기능

### 퀴즈 문제 화면 
- 간단 곱셈이 가운데 화면에 나타남
- 해당 문제가 맞으면 맞음, 틀리면 틀림 버튼을 터치
- 문제를 맞췄을 경우 각 경우에 맞게 숫자가 카운팅
- 다시 시작하기 버튼을 누르면 카운팅이 초기화 되면서 퀴즈를 다시 시작
